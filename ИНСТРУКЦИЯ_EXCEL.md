# Инструкция по построению графиков в Excel

## Шаг 1: Импорт CSV файла

1. Откройте Microsoft Excel
2. Нажмите **Файл → Открыть**
3. Выберите CSV файл с результатами (например, `results_N8_M5000_Q25.csv`)
4. Excel автоматически разделит данные по столбцам

## Шаг 2: Проверка данных

У вас должно быть два столбца:
- **Столбец A:** Размер (байт)
- **Столбец B:** Среднее время (мс)

## Шаг 3: Создание графика

1. Выделите **оба столбца** с данными (включая заголовки)
2. Перейдите на вкладку **Вставка**
3. Выберите **Точечная диаграмма** → **Точечная с гладкими кривыми и маркерами**

## Шаг 4: Настройка осей

### Горизонтальная ось (X) - Размер данных

1. Щелкните правой кнопкой по графику → **Выбрать данные**
2. В разделе **"Подписи горизонтальной оси (категории)"** нажмите **Изменить**
3. Выберите диапазон данных из столбца A (Размер в байтах)
4. Нажмите **OK**

### Вертикальная ось (Y) - Время

1. В окне **"Выбор источника данных"**
2. В разделе **"Элементы легенды (ряды)"** должен быть один ряд
3. Нажмите **Изменить**
4. В поле **"Значения Y"** выберите диапазон из столбца B (Среднее время)
5. Нажмите **OK**

## Шаг 5: Оформление графика

### Название графика
1. Щелкните по заголовку графика
2. Введите название, например:
   - "Зависимость времени передачи от размера данных (N=8, M=5000, Q=25)"

### Подписи осей
1. Нажмите на график → вкладка **Конструктор диаграмм**
2. Нажмите **Добавить элемент диаграммы** → **Названия осей**
3. Добавьте название для горизонтальной оси: "Размер данных (байт)"
4. Добавьте название для вертикальной оси: "Среднее время (мс)"

### Легенда
1. **Добавить элемент диаграммы** → **Легенда**
2. Выберите положение (например, справа)
3. Можно переименовать ряд данных на что-то осмысленное, например "Время передачи"

## Шаг 6: Улучшение визуализации

### Сетка
1. **Добавить элемент диаграммы** → **Сетка**
2. Включите основные линии сетки для обеих осей

### Формат линии
1. Щелкните правой кнопкой по линии графика
2. **Формат ряда данных**
3. Настройте цвет, толщину линии, размер маркеров

### Масштаб осей (если нужно)
1. Щелкните правой кнопкой по оси
2. **Формат оси**
3. Настройте минимум, максимум, единицы измерения

## Шаг 7: Анализ графика

Обратите внимание на:
- **Характер зависимости:** линейная, нелинейная, ступенчатая?
- **Точки перегиба:** где график меняет свое поведение?
- **Начальный участок:** как ведет себя график при малых размерах?
- **Конечный участок:** стабилизируется ли время или продолжает расти?

## Шаг 8: Сохранение

1. Сохраните файл Excel с графиками
2. Для отчета сделайте скриншот графика:
   - Щелкните по графику
   - **Копировать** (Ctrl+C)
   - Вставьте в Word документ или сделайте скриншот

## Пример интерпретации

### Для маленьких пакетов (N=8):
- График может показывать почти постоянное время для маленьких размеров (TCP overhead)
- Затем линейный рост с увеличением размера
- Возможны "ступеньки" из-за буферизации

### Для больших пакетов (N=1024):
- Более плавный линейный рост
- Меньшее влияние TCP overhead
- Четче видна пропускная способность канала

## Создание сводного графика (опционально)

Если хотите сравнить все три сценария на одном графике:

1. Откройте новый лист Excel
2. Скопируйте данные всех трех сценариев в разные столбцы
3. Создайте точечную диаграмму, выбрав все столбцы
4. Каждый сценарий будет отображен отдельной линией
5. Настройте легенду, чтобы различать сценарии

## Полезные советы

- Используйте разные цвета для разных сценариев
- Добавьте линию тренда для анализа (правый клик по ряду → Добавить линию тренда)
- Можно добавить уравнение тренда для расчета пропускной способности
- Сохраните несколько версий графика с разными масштабами для детального анализа
